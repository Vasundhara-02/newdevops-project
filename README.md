# My Devops Project

# 🚀 Self-Healing DevOps Platform

This project demonstrates a complete DevOps workflow including CI/CD, containerization, Kubernetes deployment, auto-scaling, and monitoring.

## 🔧 Tech Stack
- Node.js
- Docker
- Kubernetes (Minikube)
- GitHub Actions
- Prometheus & Grafana

## ⚙️ Features
- CI/CD pipeline using GitHub Actions
- Dockerized application
- Kubernetes Deployment & Service
- Self-healing using liveness & readiness probes
- Auto-scaling using Horizontal Pod Autoscaler (HPA)
- Load testing using BusyBox
- Monitoring with Prometheus & Grafana

## 🧪 Testing
- Load testing performed using BusyBox
- Application successfully scaled from 2 to multiple pods under load

## 📌 How to Run
1. Clone the repository
2. Build Docker image
3. Deploy using Kubernetes
4. Access service using Minikube

## 🎯 Outcome
This project simulates a real-world DevOps environment with automated deployment, scaling, and monitoring.
